# Support — Rechka AI

This is the official support repository for [Rechka AI](https://rechka.ai).

Use this repository to:
- Report a bug or unexpected behaviour
- Request a new feature or improvement
- Ask a question about how something works

## How to get help

👉 [Open a new issue](https://github.com/rechkaai/support/issues/new/choose) and choose the category that best fits your request.

Our team reviews all issues and will respond as soon as possible.

## Before you submit

- Search [existing issues](https://github.com/rechkaai/support/issues) to avoid duplicates
- Include as much detail as possible — screenshots and steps to reproduce help us resolve issues faster
- Include your company name or account ID if you have one

## Response time

We aim to respond to all issues within 1 business day.
